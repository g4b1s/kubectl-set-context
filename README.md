# Pre Req

You need to install dialog 

## RHEL
```shell
sudo dnf install dialog
```
## DEB
```shell
sudo apt-get install dialog
```
## ARCH
```shell
pacman -S dialog
```
## OSX mac
```shell
brew install dialog
```

# Use

move to any folder in your path
```shell
mv kubectl-set-context /usr/local/bin
```


```shell
kubectl-set-context
```

In dialog box select context and voila
